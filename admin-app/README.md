# admin-app

## Techs

* VueJS

## Requirements

* NodeJS: 8+
* Yarn

## Setup Instructions

* Copy environment file

  ```bash
  cp `.env.example` `.env.development.local`
  ```

* Install dependencies.

  ```bash
  yarn
  ```

* Start application in development mode.

  ```bash
  yarn serve
  ```

* For production build:

  ```bash
  yarn build
  ```
